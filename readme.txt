== Description ==

This very simple plugin allows you to disable the wp-e-commerce-dynamic style sheet without making any changes to the core files of wp-e-commerce.

Simply activate it and the stylesheet will no longer be served.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
